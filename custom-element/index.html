Ejericio

Crear un custom element de un grupo de botones, los cuales sean aceptar, cancelar y cerrar.

  1. El boton de aceptar debe mandar un CustomEvent llamado "onAccept"
  2. El boton de cancelar debe mandar un CustomEvent llamado "onCancel"
  3. El boton de cerrar debe mandar un CustomEvent llamado "onClose"

Esos eventos deben ser escuchados desde la pagina principal

Si es necesario, ocupa shadow DOM para agregar el grupo de botones
